Matrix Digital Rain

Real-time falling characters screensaver for macOS

Download GitHub

Installation

  1. Download and unzip the .saver file
  2. Open Terminal and remove quarantine:
    xattr -cr ~/Downloads/MatrixDigitalRain.saver
  3. Double-click the .saver file to install
  4. Go to System Settings > Screen Saver and select it
